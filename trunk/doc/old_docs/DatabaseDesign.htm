Database Design

 

This class is a static class designed to interface with a database for storage of user information and logs. As of yet we do not have a database and as such, it uses a static list to store users. This works across one session of the program. It will not save data from one session to the next. This aspect will be replaced by the database in the future.

 

To view details of each method run, "ant javadoc" and look at Database class.
