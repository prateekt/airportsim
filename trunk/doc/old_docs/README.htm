Click HERE for a complete step by step run-through of our system that demonstrates all of our functionality.*

You can check out login/create/delete user functionality in the first tab, and check out simulation and risk functionality in the Ground Controller tab (far right) or Local Controller tab.

The ground controller and local controller screens work properly and display the real-time positions of the airplanes. First you must click “Start Simulation”. We only have one simulation file because our system is real-time and you can control the movements of the planes so each run is a different simulation. You can move planes by clicking “Next Time Step”. If you select an action in the risk-mitigation-action columns you can execute the action by clicking the “Execute Action” button that appears. At any time you can right-click on a plane to get a list of actions for that plane. At this time it is possible to tell the plane to leave the runway and once you do you will have to reset the simulation to get it back.

 

Additionally, we have implemented A* for the go to gate action. It calculates where the plane is headed and finds the best route then instructs the plane on where to go. It does not allow planes to turn around either.  Since this feature was added very late in the game, we have not merged A* with the rest of the updates.  However, you can see the progress (which is quite far) in our A* branch on subversion.
