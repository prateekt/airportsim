Messages:

 

msgHereIsScenario(WorldState scenario) //will be a txt file to read in in future, but for skeleton, just a string.

            conditionsList.add(scenario);

 

Actions:

 

findRisk(WorldState s)

            Print(Real time conditions received: s)

            rm = new RiskMitigation

            if(2 planes are facing eachother and their velocities != 0)

                        Do(Pass risk data to Log and ATCAssistant)

                        Rm = new CollisionRisk(s.plane1, s.plane2)

                        Log.msgRiskMitigationFound(rm)

                        Atc.msgHereIsRiskMitigation(this, rm)

            If(weather != clear)

                        Rm = new WeatherRisk(s.weather)

                        Log.msgRiskMitigationFound(rm)

                        Atc.msgHereIsRiskMitigation(this, rm)

 

 

Scheduler:

 

If ($ c in conditionsList)

            findRisk(c);
