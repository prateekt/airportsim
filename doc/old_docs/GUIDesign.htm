public int validateUser(String userID, char[]password, String position)

Return type: int

Method Description: validates a user for logon to system.

Method Preconditions: userID must exist, passwords must match, employee must have selected a position that they have been approved for.

Method Postconditions:  User is validated and an integer is returned for appropriate logon rights.

Method Interaction: no other methods

Input and Output Format: Input taken for userID, password, and position type.  If these match database of treemaps, then user is taken to appropriate user panel.

Test Descriptions:  Logon as authorized type of user.  Appropriate panel will be displayed.  Logon as unauthorized type of user.  Error message will be displayed.
